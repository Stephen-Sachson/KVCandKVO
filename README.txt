This Demo code describes how to use KVC mechanism to set and get value to and from another object and use KVO to observe value change of another object and then deal with it.

1.Input book name, author and price and tap Add.
2.Repeat step 1 as many times as you want.
3.Tap Show to show informations about books you have added.(via KVC)
4.Tap Chanllenge for 20% discount to decrease the price of each book.
Message will be presented as the prices decreased.
